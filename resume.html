Work history

Used.ca - Full Stack Developer

At Used.ca I created a tool for the administration of advertisements for use by employees, and assisted with a the development and implementation of a new codebase. The administrative tool was created using Flask, MongoDB, on an ELK stack for simple searching and logging of advertisements. The new codebase was build using primarily Django.

Eventbase - QA Analyst

My work at Eventbase was focused on ensuring the quality of the applications that were being delivered to events like SXSW, and CES. Testing was done in Android, iOS, and Windows Phone environments.

University of Victoria - Research Developer

I developed a language processing framework to process encyclopedia entries from the encyclopedia of Modernism, and a web-front to display the relations. The encyclopedia entries needed to be classified into ontological categories and relationships between entities. Using the StanfordNLP library and various Python programs the analyzed entities were deposited into an RDF database. The relations between entities are then viewable and compared on the web front end, linkedmods.uvic.ca.

Projects

Scalable Stock Sales System

This project is a scalable stock sales system interacting with a legacy quote server. The project was built in golang using Docker for deployment, the gorilla package for webserver routing, and mongodb for storage. The project uses RPC for simplified execution and increased transactions per second.

Innotime

This was a brief project during a programming competition designed for maximizing time management. As a group we implemented note taking, reminders, and calendar functionality using jQuery. We used localstorage as a challenge for all of our data to make the project persistent without internet.

Trumpiness

This project used machine learning on a retrieved twitter dataset to classify the tweets of Donald Trump, Ellen Degeneres, Elon Musk, and Barack Obama. Once classified, a user is able to run example tweet text, or retrieve tweets by id to analyze against the classifier to see how alike their tweet is to our classification.
