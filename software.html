One of my larger projects is the Linked Modernisms project. It's a website that helps visualize connections between various characters in the field of Modernism. The coolest part of it was getting to work with the Stanford Natural Language Processing software. Since the project is associated with an Encyclopedia of Modernism, we had access to a large volume of text to train with. The project also gave me a chance to learn a lot about web development. I had never used used html, css, or javascript extensively in an actual application prior to this project.
